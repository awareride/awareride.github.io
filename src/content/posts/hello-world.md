## Why AwareRide?

We started AwareRide because we believe the best tools are built with **awareness** — of context,
of people, and of the systems we all share.

The software industry is obsessed with speed: faster builds, faster deployments, faster
frameworks. But speed without understanding creates fragility. We want to build tools that help
developers **understand** complexity before they optimize it.

## What We're Working On

### Packscope

Our first tool is [Packscope](/packscope) — a Node CLI that unpacks a single shipped JavaScript
bundle into a navigable, executable project tree.

If you've ever stared at a minified 21 MB webpack bundle and wondered "what's actually in here?",
Packscope is for you. It extracts every module into its own file, preserves the original runtime
so the unpacked tree still runs, and even lets you edit modules and rebuild the bundle.
