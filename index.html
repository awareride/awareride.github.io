Our Focus

Perception, space, and wellbeing

We believe the best tools are built with awareness—of context, of people, and of the systems we all share. Our work sits at the intersection of cognitive science, spatial computing, and developer experience.

Projects

Tools we are building

Node CLI JavaScript

packscope

Unpack a single shipped JavaScript bundle — from webpack, rspack, rollup, esbuild, or Vite — into a navigable, executable project tree. Read, grep, edit, and rebuild modules with confidence.

View Repository Documentation 
$ npx packscope ./dist/app.js ./out

# webpack/rspack output
out/
├── header.js
├── webpack-runtime.js
├── runtime.js
├── index.js
├── modules/<id>.js
├── manifest.json
├── rebuild.js
└── package.json

$ node out/index.js --version
2.106.4

Principles

How we build

  • Open by default Source code, decisions, and roadmaps are shared with the community.
  • Human-first We optimize for clarity, accessibility, and calm over novelty for its own sake.
  • Interdisciplinary We draw from design, cognitive science, and systems engineering.
  • Carefully measured From bundle sizes to cognitive load, we believe in understanding before optimizing.
