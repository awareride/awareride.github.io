# awareride.github.io

The GitHub Pages site for [AwareRide](https://github.com/awareride), served at
**https://open.awareride.com**.

## About

AwareRide is an open-source initiative exploring human perception, spatial intelligence, and
wellness software.

## Projects

- **[packscope](https://github.com/awareride/packscope)** — A Node CLI that unpacks mono
  JavaScript bundles from webpack, rspack, rollup, esbuild, and Vite into navigable,
  executable module trees. [Docs](https://open.awareride.com/packscope)

## Development

This is a static site built with plain HTML, CSS, and a small amount of vanilla JavaScript. It is
automatically deployed by GitHub Pages.
